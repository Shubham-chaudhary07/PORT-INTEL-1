Port-Intel is a real-time cybersecurity honeypot platform that captures, analyzes, and visualizes attack activity to help students and beginners understand real-world threats.
Project Structure

app.py – Flask backend server, API routes, and process control

honeypot.py – SSH honeypot listener that captures real attack attempts

att.py – Attack simulation engine for generating test attack logs

log.json – Stores all captured and simulated attack events

index.html – Web dashboard UI

style.css – Dashboard styling

script.js – Frontend logic, charts, and API calls
